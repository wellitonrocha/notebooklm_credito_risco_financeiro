# notebooklm_credito_risco_financeiro
Mini guia de estudo sobre Análise de Crédito e Risco Financeiro utilizando IA Generativa no NotebookLM. O projeto reúne curadoria de fontes, engenharia de prompts, resumos estruturados, glossário financeiro e reflexões críticas, demonstrando o uso da IA como ferramenta de aprendizagem ativa.
